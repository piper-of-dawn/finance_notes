Trade in Intermediate Inputs

  • Price of the final good: $$ p_n = \theta_{n1} \ p_1 + \theta_{n2} \ p_2$$
  • Fall in the price of first good \(\Rightarrow \) p_1 - p_2 < 0
  • $$\therefore p_1 < p_n < p_2 $$ Price of thefinal good is between the prices of inputs.

    • Continuum of Outputs

    Let \(z \in [0.1]\) denote the many activities undertaken in industry.This set represents set of intermediate goods.

    \(x(z)\: Quantity produced of each one of these inputs.

    \(a_H(z)\) is the unit requirement of skilled labour.\(a_L(z)\) is the unit requirement of un-skilled labour.

    \(a_H(z) \over a_L(Z)\) is non decreasing in Z. Last activity is most skill-intensive.

    Production Function

    $$ x(z)=A\left[\min \left\{\frac{L(z)}{a_{L}(z)}, \frac{H(z)}{a_{H}(z)}\right\}\right]^{\theta} K^{1-\theta}, \quad z \in[0,1] $$
    Final good production function is Cobb-Douglas production function over all inputs.

    $$ \ln Y=\int_{0}^{1} \alpha(z) \ln x(z) d z, \quad \text { with } \int_{0}^{1} \alpha(z) d z=1 $$

    Firms doing the assembly will wish to source the inputs from the minimum cost location.

    Assumption on factor prices is: $$ \frac{q}{w} < \frac{q^{*}}{w^{*}}, \quad \text { and } r < r $$ Note that home country is a rich country.

    Deciding the minimum cost location of each inputs.
