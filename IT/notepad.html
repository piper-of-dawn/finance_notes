Factor Price Insensitivity

There cannot be a FPI lemma in a closed economy. FPI says that factor endowments can increase without affecting the factor prices. $$ =\frac{\delta_{k k}}{s_{k}}+s_{k}-1, \quad \text { if } k=l \\ =\frac{\delta_{k l}}{s_{k}}+s_{l}, \quad \text { if } k \neq l $$ Change in Endowment of labour has no impact on change of capital.

We get these two expressions for \(frac{\partial \ln w_{k}}{\partial \ln V_{l}}\)

There are two cases here: \(\delta_{kk}\) is the regression coefficient.

\begin{array}{l} \frac{\partial \ln w_{k}}{\partial \ln V_{l}}=\frac{\partial \ln \left(s_{k} G / V_{k}\right)}{\partial \ln V_{l}}=\frac{\partial \ln s_{k}}{\partial \ln V_{l}}+\frac{\partial \ln G}{\partial \ln V_{l}}-\frac{\partial \ln V_{k}}{\partial \ln V_{l}} \\ =\left(\frac{\partial s_{k}}{\partial V_{l}}\right)\left(\frac{V_{l}}{s_{k}}\right)+\left(\frac{\partial G}{\partial V_{l}}\right)\left(\frac{V_{l}}{G}\right)-1, \quad \text { if } k=l \\ =\left(\frac{\delta_{k k}}{V_{l}}\right)\left(\frac{V_{l}}{s_{k}}\right)+\frac{w_{l} V_{l}}{G}-1 \\ =\frac{\delta_{k k}}{s_{k}}+s_{k}-1, \quad \text { if } k=l \\ =\frac{\delta_{k l}}{s_{k}}+s_{l}, \quad \text { if } k \neq l \end{array}
