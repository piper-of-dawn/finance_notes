Defining Event Studies

Event studies are used to assess the financial impact of changes in corporate policy.​ This change could be anything. It could be, corporate control changes, corporate refocusing, CEO turnover, layoffs, plant closures, frauds, strategic investment decisions (merger acquisitions), earnings announcement.

They attempt to measure the effects of an economic event on the value of firms.​

What is an event?

Tt should be an unanticipated occurance to investors. Hence events are associated with an "abnormal" stock price effect.

Why Event Studies?

In principle, the accounting-based measures of profits can be used to measure the performance of firms. But there was a seminal paper by Fisher and McGowan in 1983 which criticized the misuse of accounting-based measures of profits to infer the monopoly profits.

$$\text{Accounting rate to return} = {\text{Profits} \over \text{Assets}}$$

Stock prices on the other hand are not subject to manipulation by insiders and are supposed to reflect the true value of firms.​ There exists a substantial amount of literature on earnings management or what is called window dressing. Window dressing implies that the people with insider information about the company that is the managers of the company, fudge the accounts to manipulate stock prices.

However we will assume that such problems do not exist. This serves as a base assumption of event studies that stock prices reflect the true value of firms. Consequently, an events economic impact can be measured using stock prices.​

How to study events?

  • Identify the event
  • Identify the event and estimation window timelines

    • Usually, the event window is defined such that it is larger than the specified period of interest.It is easy to implement as all as the only data that is required is names of firms, event dates and stock prices.​

    Defining Sample Space

    We mainly proceed with the listed firms because the stock prices of those firms are observable. This also kind of introduces a sampling bias in our event study. Even in listed firms, we need to set up a criterion because there might exist some firms for which, trading does not take place for many days. There might be some issues with regard to the industry representation as well because there might exist some industries where events like mergers and acquisitions are very common and vice versa.

    Defining Abnormal Return

    Appraisal of the impact of the event requires a measure of abnormal return. The abnormal return is the actual ex-post return of the security over the event window minus the normal return of the firm over the event window. In simple words, the difference in the returns that the event has created in returns on stocks ​is called an abnormal return. It can only be seen after the event has happened and hence ex-post. So for assessing the abnormal return there should be a definition for normal return.

    Defining Normal Return

    The normal return is defined as the expected return without conditioning on the event taking place. We can use the market model to estimate this normal return such as standard CAPM. The most common choice for an estimation window is using the period prior to the event window. For example, in an event study using daily data and the market model, the market model parameters could be estimated over the 120 days prior to the event.​

    Estimation Procedure

    Step 1: Defination of windows

    Define \(\tau = 0\) as the event date.

    Let \(L_1 = T_1 - T_0\) be the length of the event window.

    Let \(L_2 = T_2 - T_1\) be the length of the estimation window.

    Let \(L_3 = T_3 - T_2\) be the length of the post event window.

    Event Window

    Event window is the small time period after the event has occured.

    Estimation Window

    Estimation window is the long time period before the event has occured.

    Post Event Window

    Post Event window is the long time period after the event has occured.

    There will also be serial correlation. The estimation window \(L1\) should be large.

    Step 2: Cumulate Abnormal returns over time as well as firms

    Define \(CAR(\tau_1,\tau_2)\) as \(\sum AR_{it}\). Aggregate all the abnormal returns between two events.

    The distribution of the cumulative abnormal return under \(H_0\) is \( N (0,\sigma^2_i(\tau_1,\tau_2)) \). The abnormal return observations must be aggregated across the event window and across observations of the event. ​ The individual securities abnormal returns can be aggregated for each event period \(\tau = T_1 \ldots T_2\). Given \(N\) events, the sample aggregated abnormal returns for period \(\tau\) is: \( AR_\tau={1 \over N} {\sum AR_{i\tau}}\) and for large \(L1\) its variance is ​ $$Var (AR_\tau)={1 \over N^2} {\sum \sigma^2_{\epsilon i}}$$

    Step 3: Test the hypothesis

    One can now test the null hypothesis that the abnormal returns are zero.​

    Note that null hypothesis here is that no abnormal return should be significantly different from 0. If it turns out significantly different from 0, then the event has a consequence.Note that if the null hypothesis is true, then the event has been anticipated. So ultimately what we are driving towards is the test of differences.

    If the markets are efficient the difference between cumulated returns and the average returns should not be greater than 0. Market Efficiency is the baseline assumption here.

    For the variance estimator the assumption that the event window of the N securities do not overlap is used set the covariance terms to zero.​ This means that events need to be distributed over time in such a way that event windows do not overlap that is events should occur simultaneously. Because if they overlap the covariance terms won't be 0.

    In practice, because \(\sigma^2_{\epsilon_i}\), is unknown, an estimator must be used to calculate the variance of the abnormal returns. The usual sample variance measure from the market model regression in the estimation window is an appropriate choice.​

    One common modification to the above approach is to standardize each abnormal return using an estimator of its standard deviation. Such standardization can lead to more powerful tests.​

    Rethinking, What is an event?

    Fama says that event studies are the best tests for market efficiency i.e prices reflect the complete information. Hence the definition of event is as follows:

  • To the extent that an event is uncertain and unanticipated, it will generate abnormal returns.
  • Once the information about the event becomes public the investors would immediately act upon the information and it would be subsumed into the stock prices.

    • Therefore an event is anything that generated new information that makes the market deviate from effciency. So till the null hypothesis is accepted, the event study acts as a testament for the theory of market efficiency. But when the \(H_0\) is rejected we actually don't know where the problem is. There could be anomalies that is markets are not efficient or normal returns as calculated by CAPM is wrong.

    Event windows and Market Efficiency

    If markets are efficient, the event windows should be reasonably small, because when we assume efficiency we apriori assume that investors will act upon any new information as soon as it becomes public.

    If the effects of events are not realised within a short period of time it is the obligation of the researcher to explain this.

    Measurement, Data and Robustness issues

    Issue of Confounding Effect

    This means that one event impacts another event. So researcher has to isolate the impact of other events over that event. Hence controlling the impact of confounding event is very necessary.

    Issue of Granularity

    The question of what granularity of time series data should a event study use is not trivial question. Most event studies have used closing price of stocks. Using a less granular data will reduce the power of the test whereas using highly granular data will end up modelling noise rather than signal.

    Issue of Cross-sectional variations

    Each event might impact different firms in different ways. Can we explain those cross-sectional differences?

    Event studies have been successful is explaining many theoretical results such as why returns fall when there is an equity issue.

    When firms issue debts, the abnormal returns are closer to zero. Debt will not have an influence because if firm can manage to raise debt then it signals a good financial health of the firm.

    Pecking order theory of Capital Structure

    First things that firms do is raise internal finance and then subsequently go on to external financing.

    Roseff and Zaman (1988)

    Roseff and Zaman have found strong evidence that strong form of market efficiency does not hold. There is an evidence of rampant insider trading and people who mimic insider traders.
