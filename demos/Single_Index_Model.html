Single Index Model

Definitions:

The return on a stock can be written as: $$ R_i = a_i + \beta_i.R_m $$ where:

  • \(a_i\) is the component of security i’s return that is independent of the market’s performance — a random variable.
  • \(R_m\)s the rate of return on the market index — a random variable.
  • \(\beta_i\) is a constant that measures the expected change in \(R_i\) given a change in \(R_m\).
  • \(\sigma_m \) denote the standard deviation of \(R_m\).
  • \(\sigma_{ei} \) denote the standard deviation of \(i_{th}\) security.
Let us break \(a_i\) into the intercept \(\alpha_i\) and the random component \(e_i\). \(a_i = \alpha_i + e_i\). Now the return on a stock can be written as: $$ R_i = \alpha_i + \beta_i.R_m + e_i$$
