Conditional Least Squares

$$ X_t = \alpha_1 X_{t-1} + \alpha_2 X_t-2 +... \\[10pt] \epsilon_t = X_t - \alpha_1 X_{t-1} - \alpha_2 X_t-2 +... \\[10pt] \sum_{t=p+1}^T = \sum ( X_t - \alpha_1 X_{t-1} - \alpha_2 X_t-2 +...)^2 \\[10pt] SS = f(\alpha_1, \alpha_2, \alpha_3....) $$

\(X_1\) has to be treated as given because it is a evolving time series.

Minimise over \(\alpha_1, \alpha_2, \alpha_3.... \alpha_p\) \epsilon_t = X_t - b_{t-1} \epsilon_{t-1} Hence, \epsilon_2 = X_2 - b_{1} \epsilon_{1} \epsilon_3 = X_3 - b_{2} \epsilon_{2} ans so on on... In MA(q) process, first \(q\) \(\epsilon\)s are to be set to 0.
